How to include JavaScript in HTML?
 We can include Javascript in HTML in 2 ways they are
  i.Internal JavaScript
  ii.External JavaScrpt
i.internal JavaScript :We can include JavaScript using 'script' as element
 eg:
   <script>
      // here you can write the JavaScript
   </script>
ii.External JavaScript: we can include external JavaScript file .js file to HTML using 'script' and src atribute;
  eg
  <script src="Path of the script"></script>
